# selfdev-timer
**지속 가능한 성장을 위해, 자기개발 시간을 기록하고 시각화하는 프로젝트**

<img width="482" height="626" alt="Image" src="https://github.com/user-attachments/assets/268b806d-c657-46f8-882b-6e4c41e8a919" />

## How To Use
- 자기개발/휴식 타이머
	- 시작, 일시정지, 정지 버튼으로 타이머 제어, 누적 시간은 60초마다 갱신
- 시간 보정
	- 출퇴근 중 자기개발을 하거나 타이머를 실수로 정지시키지 않았을 때 사용하는 수동 보정 기능
- 색상 기준, 이전 기록
	- 이전 기록에 표시되는 시간 색상 기준값을 변경 가능
	- 오전 6시를 기준으로 날짜 변경
